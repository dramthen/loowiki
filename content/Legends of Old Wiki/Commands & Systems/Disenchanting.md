The disenchanting table found in the [[Nexus]] allows you to take three unwanted [[Equipment|equipment]], or two plus a [[Shards|shard]] to generate new random [[Equipment|equipment]]. The generated item is slightly weighted based on input and is scaled within your characters level - not the average level of the input items.

Using completely random [[Equipment|equipment]] will generate a completely random output.

Using at least two [[Equipment|equipment]] of a kind and any third will output the same type. For example two weapons and a ring, will output a random weapon. Currently it is not possible to input two swords and get a sword output, instead it will only ever generate a random weapon, though this is planned for improvement.

Currently it is possible to force generate an accessory, a weapon (shields are considered weapons), or an armor piece.

A [[Shards|shard]] can be used in leu of a third [[Equipment|equipment]] piece. The type of shard will force a stat onto an item in addition to all other randomly generated stats. See the [[Shards]] page for more details.
